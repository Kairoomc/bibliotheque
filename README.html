📚 Application de Gestion de Bibliothèque

Application Java complète pour gérer une bibliothèque de livres avec interface graphique moderne.

🖥️ Version Desktop avec Java Swing et design moderne via FlatLaf

--- ## ✨ Fonctionnalités ### 🔄 Fonctionnalités générales - ➕ Ajouter des livres - 🔍 Rechercher en temps réel - 🗑️ Supprimer un livre - 📋 Afficher la liste des livres - 💾 Sauvegarde persistante automatique - 🖥️ Interface utilisateur moderne et responsive - 🧼 Code propre, structuré et commenté ### 🖥️ Version Desktop (Java Swing + FlatLaf) - 🎨 Interface sombre ou claire avec thème FlatLaf - 🔍 Recherche dynamique par titre - ✅ Application exécutable en `.jar` - 🧠 Conception orientée objet (MVC) --- ## 🛠️ Technologies utilisées | Plateforme | Technologies | |-------------|--------------------------------------------------| | Desktop | Java 17+, Swing, FlatLaf, MVC | ---
